treecoreset
===========

To compile you must have:

 1. python2 (with numpy, sklearn installed)
 2. ctypesgen ([download](https://code.google.com/p/ctypesgen/)) located in the same directory as the code.
 3. gcc and stuff

To test type `make` and then run iris.py
